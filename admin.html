Админка состава

Вход, редактор и пуш в репу

Логин и пароль открывают интерфейс. Реальная запись на сайт делается либо скачиванием roster.json, либо через GitHub API с токеном владельца репозитория.

Без серверной магии

Пароль хранится как PBKDF2-хеш. Это нормальная защита от случайных глаз, но не банковский сейф: статический сайт нельзя сделать полноценной закрытой админкой без бэкенда.

Токен вводится руками и не лежит в коде сайта. Без него кнопка пуша не сможет изменить репозиторий.

Не вставляй токен в код и не коммить его в репу. Это будет уже не админка, а приглашение вынести всё нахрен.
