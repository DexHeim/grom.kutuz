Редактор состава

Локальная админка без бэкенда

GitHub Pages не даёт серверную базу данных, поэтому этот экран делает черновик состава в браузере и экспортирует готовый roster.json. Потом файл надо заменить в папке data и закоммитить.

Важно

Это не защищённая админка с логином. Любой может открыть страницу, но чужой человек не сможет изменить сайт без доступа к репозиторию.

Черновик

Редактирование data/roster.json

Заполни форму, скачай JSON и замени им файл data/roster.json в репозитории. Это тупо, но надёжно и без дыр с токенами в открытом сайте.

Ручное редактирование

Как опубликовать изменения

  • Нажми «Скачать» и получи файл roster.json.
  • В репозитории открой папку data.
  • Замени старый roster.json новым файлом.
  • Сделай commit. После обновления Pages сайт подтянет новый состав.
  • Доступ на редактирование репозитория выдаётся через GitHub Collaborators.
