# ОСН «Гром» — сайт для GitHub Pages

Статический сайт без сборки: HTML + CSS + JS. Можно загрузить файлы в репозиторий и включить GitHub Pages.

## Что внутри

- `index.html` — структура сайта: FAQ, правила, иерархия, ссылки, памятки, повышение, калькулятор и рапорт.
- `assets/css/style.css` — стиль, адаптив, тёмная/светлая тема.
- `assets/js/app.js` — баллы, доказательства, система повышения, логика калькулятора, автогенерация рапорта.
- `assets/img/raid-map.png` — карта налётов и ограблений.
- `assets/img/arrest-memo.png` — памятка по задержанию.
- `assets/img/calculator-reference.png` — исходный скрин таблицы как референс.
- `assets/img/favicon.svg` — иконка сайта.
- `.nojekyll` — отключает Jekyll-обработку на GitHub Pages.

## Быстрый запуск локально

Открой `index.html` двойным кликом в браузере. Калькулятор работает без сервера.

## Как залить на GitHub Pages

1. Создай репозиторий на GitHub.
2. Загрузи **содержимое** этой папки в корень репозитория: `index.html`, `.nojekyll`, `assets/`, `README.md`.
3. Открой репозиторий → **Settings** → **Pages**.
4. В блоке **Build and deployment** выбери **Deploy from a branch**.
5. В **Branch** выбери `main` и папку `/root`, затем нажми **Save**.
6. Через пару минут сайт будет доступен по адресу вида `https://username.github.io/repository-name/`.

## Где править

### Баллы и доказательства

Файл: `assets/js/app.js`

Найди массив:

```js
const activities = [
```

Строка активности выглядит так:

```js
{
  id: 'patrol',
  title: 'Патруль (30 минут)',
  points: 50,
  proof: 'Скриншот начала патруля, каждые 10 минут и окончания патруля'
}
```

Меняешь `title`, `points` или `proof` — таблица и рапорт обновятся автоматически.

### Система повышения

Файл: `assets/js/app.js`

Найди массив:

```js
const promotionSteps = [
```

`points: null` значит, что у ступени нет балльного порога, только обязательные условия.

### Ссылки

Файл: `index.html`

Найди блок:

```html
<section class="section section-pad" id="links">
```

Там лежат форум, законка, Google Docs, YouTube, Discord и таблица Google Sheets.

### Картинки-памятки

Замени файлы, сохранив имена:

- `assets/img/raid-map.png`
- `assets/img/arrest-memo.png`

Так HTML менять не придётся.

### Цвета

Файл: `assets/css/style.css`

В начале файла лежат переменные:

```css
:root {
  --bg: #07111f;
  --primary: #58a6ff;
  --accent: #79ffe1;
}
```

Их можно менять без копания по всему CSS.

## Важно перед публикацией

Проверь актуальность Discord-ссылок, форумных тем и требований по повышению. На сайте уже внесены данные из FAQ ОСН «Гром» и системы повышения, но регламенты в РП-проектах меняются быстрее, чем люди успевают сказать «а кто старший группы?».
