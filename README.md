## Состав

Основной источник состава — Google Sheets:

`https://docs.google.com/spreadsheets/d/1e4rEXR7FlKvlLoiWK41Aer0tLpqLuhXVbyrpipALhBQ`

`assets/js/roster.js` сначала пробует загрузить публичную таблицу через Google Visualization API. Если таблица недоступна, сайт автоматически откатывается на `data/roster.json`.

Чтобы состав обновлялся на сайте без правок кода, таблица должна быть доступна хотя бы на просмотр по ссылке.

Ожидаемая структура колонок в таблице:

- `A:B` — имя и фамилия;
- `C:D` — статистический ID, на сайте выводится как «Жетон»;
- `E:F` — позывной;
- `G:H` — должность;
- `I` — звание;
- `J` — примечание.

## Статусы состава

Поддерживаются статусы:

- Активен
- Стажёр
- Резерв
- Отпуск
- Вакантно

Статус определяется из данных строки таблицы. Если в строке встречается `Вакантно` / `Вакант`, карточка получает статус «Вакантно».
